Dans ce fichier .skill :
Le SKILL.md principal avec le workflow complet en 6 étapes (collecte profil → recherche par indicateur → rapport structuré → analyse d'impact par poste → email → intégration Cowork si dispo), plus 3 fichiers de référence :

email-template.md — le template email équipe avec variante courte (Slack/Teams)
sources-catalogue.md — toutes les sources classées par indicateur ET par secteur d'OF (santé, BTP, numérique, management, insertion...)
impact-matrix.md — qui reçoit quoi selon son poste, grille de priorisation automatique 🔴/🟡/🟢, et règles de qualification d'une actualité

Comment l'utiliser :
Dis simplement "prépare la veille de cette semaine" ou "rapport de veille I23-I26" et la skill se déclenche. Elle te demandera le profil de l'OF si elle ne le connaît pas déjà, puis lancera les recherches et produira le rapport + l'email dans la foulée.
Sur Cowork : si tu as Claude Cowork installé, la skill peut alimenter automatiquement ton tableau de veille (Notion ou Airtable) — je peux développer ce module si tu veux l'activer.
